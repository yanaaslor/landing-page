# landing-page
This project is a responsive landing page built as part of The Odin Project curriculum. 
It focuses on practicing core HTML and CSS skills.

The design is based on a provided reference layout and aims to replicate it as closely 
as possible. The project includes multiple sections such as a hero, information, quote, 
and a call-to-action banner.

Through this project, I strengthened my understanding of:

    - CSS Flexbox layout
    - CSS Fundamental Properties
    - Page structure and semantic HTML
